# 2019 review:

## Draft Mechanism
I think we did this draft in a Google Sheet. This year was different because instead of drafting schools, we drafted states, so you got all of the schools in the state.

## Scoring
Done with a C# app. Source code available at https://github.com/zo0o0ot/2020-draft-scorer

## Document overview
- nflDraft-2019.csv : The results of the NFL draft. Points and bonuses for trades are in separate columns in this CSV.
- stateResults-2019.csv : Results for the states, which were the drafted positions in this year's draft. It shows predicted points versus actual points, and I think it leaves out the trade bonuses since they were calculated separately.
- draftPoints-2019.csv : This shows the end results for the NFL draft from the owners perspective. The points and bonuses are separate so it would be necessary to combine them for the final standings.

## Historical reference
