# 2020 review:

## Draft Mechanism
I think we did this draft in a Google Sheet.

## Scoring
Done with a C# app. Source code available at https://github.com/zo0o0ot/2020-draft-scorer

## Document overview
- auctionDraft-2020.csv : The original auction results. A Variety of fields including draft performance: School,Person,Price,Projected Points,Projected Points/$,Actual Points,Actual Points/$,Difference/$,Difference
- nflDraft-2020.csv : The actual NFL draft including the school and score for each pick
- leagifyResults-2020.csv : I created a csv with numerical scores for each team for each pick of the draft, so that it could be used as a multi-color line chart to show how each player scored over the course of the draft.
