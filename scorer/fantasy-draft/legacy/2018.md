# 2018 review:

## Draft Mechanism
I think we did this draft in a Google Sheet. This year was different because instead of drafting schools, we drafted states, so you got all of the schools in the state.

## Scoring
I see no source code listed for scoring, so I think that this scoring was done directly in the Google Sheet

## Document overview
- stateResults-2018.csv : Results of states compared to predictions, including the owner of each state
- stateDraft-2018.csv : The results of the state draft, showing the order that the states were picked, the region, and predicted performance
- draftResults-2018.csv : The actual picks in the NFL draft, showing the picks, the school, the state, the owner, the value, and if there was a trade bonus for the pick.
- standings-2018.csv : Final standings for each owner in the draft. Jawad won by 1 point. Shows Owner,Total Points,Drafted Players,Total Trade Bonuses,Player Value
