# 2017 review:

## Draft Mechanism
I think we did this draft in a Google Sheet. This year was not an auction, it was a snake draft.

## Scoring
I see no source code listed for scoring, so I think that this scoring was done directly in the Google Sheet

## Document overview
- nflDraft-2017.csv : Results of NFL draft along with Leagify point values. Does not correlate schools to owners.  Notably, in this year, undrafted free agents are also included as pick 254, which is one pick after the last pick, and each of those players is worth 1 point. In the CSV is Round,Pick,Team,Player,Position,School,Value,Traded,Trade Bonus,Total Points,Total Combined Points,Trade Info.
- leagifyDraft-2017.csv : The results of the snake draft for schools. Lists School,Owner,LeagifyRound,LeagifyPick,LeagifyPosition
